### Added
- Explain `-Denv=dev`.
- Add support for local configuration of `proxy.use.rules`.

### Changed
- Support for map.apps 4.6.0.
- Update ct-mapapps-gulp-js version.
- Use maven.home not M2_HOME.
- Update node, npm and dependencies.
